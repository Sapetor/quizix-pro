Cómo funciona

Cuatro pasos del silencio al aula conectada.

Diseñado para que un profesor abra su laptop, proyecte el PIN y tenga a toda la clase jugando en menos de sesenta segundos.

01

Crea o genera tu quiz

Escribe tus preguntas, sube un CSV, o deja que Claude o un modelo local las genere desde tu material de clase.

02

Proyecta el PIN

Un código de 6 dígitos y un QR. Los estudiantes se unen desde cualquier dispositivo en la red — sin apps, sin cuentas.

03

Juega en vivo

Temporizadores sincronizados, puntuación por velocidad, y tablero de posiciones proyectado para toda la clase.

04

Analiza y exporta

PDF, Excel, gráficos por pregunta. Compara sesiones del mismo quiz entre cursos.

When ON, faster answers earn more points (up to +1000 for easy, +2000 medium, +3000 hard). When OFF, all correct answers earn flat base points regardless of speed.

Manim Animation Tutorial

What are Manim Animations?

Manim lets you create mathematical animations using Python code. Attach animations to your quiz questions to make concepts visual and engaging.

How to Use

  1. Question Animation tab — plays alongside the question during the quiz
  2. Explanation Animation tab — plays after the answer is revealed
  3. Write Manim Python code in the editor
  4. Click Render Animation and wait a few seconds
  5. Preview the result, then save your quiz

Code Examples

Click any example to copy it to your clipboard.

Basic Shapes
from manim import *

class ShapesScene(Scene):
    def construct(self):
        circle = Circle(color=BLUE, fill_opacity=0.5)
        square = Square(color=GREEN, fill_opacity=0.5)
        shapes = VGroup(circle, square).arrange(RIGHT, buff=1)
        self.play(Create(circle), Create(square))
        self.play(Rotate(square, PI/4))
        self.wait(1)
Text & Annotations
from manim import *

class TextScene(Scene):
    def construct(self):
        title = Text("Key Concept", font_size=48, color=YELLOW)
        self.play(Write(title))
        self.play(title.animate.to_edge(UP))

        bullet1 = Text("- Point one", font_size=28)
        bullet2 = Text("- Point two", font_size=28)
        bullets = VGroup(bullet1, bullet2).arrange(DOWN, aligned_edge=LEFT)
        self.play(FadeIn(bullets))
        self.wait(1)
Number Line
from manim import *

class NumberLineScene(Scene):
    def construct(self):
        line = NumberLine(x_range=[-5, 5, 1], include_numbers=True)
        self.play(Create(line))

        dot = Dot(line.n2p(3), color=RED, radius=0.15)
        label = Text("x = 3", font_size=24, color=RED)
        label.next_to(dot, UP)
        self.play(Create(dot), Write(label))
        self.wait(1)
Graph / Function Plot
from manim import *

class GraphScene(Scene):
    def construct(self):
        axes = Axes(x_range=[-3, 3], y_range=[-1, 9], axis_config={"include_numbers": True})
        graph = axes.plot(lambda x: x**2, color=BLUE)
        label = axes.get_graph_label(graph, label="x^2")
        self.play(Create(axes), Create(graph), Write(label))
        self.wait(1)

Tips

  • Keep animations short (2–5 seconds) for quiz flow
  • Use self.wait(1) to pause between steps
  • Your class must extend Scene (e.g. class MyScene(Scene))
  • Always start with from manim import *
  • Use Text() for labels — MathTex() requires LaTeX installed
